CSS

DEFINIÇÕES GERAIS

O que é

CSS (Cascate Style Sheet) são documentos utilziados para impregar estilos e personalização em uma página web. Enquanto o HTML é a linguagem de marcação, ou seja, o esqueleto que define o site, o CSS é a formatação do layout e aparência do mesmo.

Como Aplicar

O CSS pode ser aplicado de três maneiras diferentes: inline, incorporado e folha de estilo externa. A Inline é a forma de aplicação do CSS direto no elemento HTML, onde é criada a propriedade style="", e atribuído o conteúdo CSS dentro das aspas duplas. Já a incorporada, é quando se cria uma tag HTML, ao final do código HTML mas antes de seu fechamento, chamada style /style, e dentro dela é escrito o código CSS. Ela já funciona melhor do que a Inline. Já a última, e recomendada forma, é a de folha externa, onde é criado um arquivo .css e este arquivo é importado em todas os arquivos HTML que quiserem implantar o seu estilo determinado. Sua importação é dada pela tag link rel="stylesheet" href="css.css"

ESTRUTURA CSS

Sintaxe

A regra CSS é da seguinte forma: p { background-color: red; }, onde "p" é o nome do elemento HTML a ser alterado, dentro de chaves irá o conteúdo CSS de fato. E dentro destas chaves vem a propriedade css seguida de dois pontos e o valor desta propriedade, isto sendo conhecido como declaração CSS.
