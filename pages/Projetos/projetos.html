OS PROJETOS

Projeto de Sala

O projeto de sala será um portal inspirado na página inicial do Google, onde terá uma barra de pesquisa para pesquisar o nome específico de cada atividade e logotipo centralizados, logo abaixo os atalhos para as 6 primeiras atividades e o link para o projeto em Angular.
Este site terá rolagem infinita e na sua segunda parte irei descrevendo as atividades executadas, com uma âncora ligando cada atalho ao seu referido resumo.

Projeto em Angular

O projeto em Angular será uma landign page para um site de reservas de quartos de hotéis. É o reaproveitamento de uma ideia já antes feita em CSS, HTML e JS puros.
